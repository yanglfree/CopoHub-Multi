隐私政策

我们重视您的隐私与数据安全。本政策适用于 CopoHub（GitHub 探索者）应用及其落地页，说明我们如何处理与您相关的信息。

一、我们收集的信息

为提供核心功能，我们可能处理以下信息：

  • 本地缓存数据：如您的应用设置、主题偏好、缓存的 GitHub 数据等，主要保存在您的设备本地。
  • 意见反馈信息：当您主动提交反馈时，我们会接收您填写的反馈内容，以及可选的联系方式、客户端类型、应用版本等信息。
  • 订阅与交易信息：当您购买 Pro 会员时，相关订单状态由应用商店平台返回，用于校验和恢复会员权益。
  • 必要技术信息：在请求 AI 分析与数据接口时，服务端可能记录基础日志（如请求时间、接口状态）用于运维与安全保障。

二、信息使用目的

我们仅在必要范围内使用信息，用于：

  • 保障数据获取、AI 分析等核心功能正常运行；
  • 处理反馈问题、定位故障并持续改进产品体验；
  • 完成会员权益开通、订阅恢复与相关服务支持；
  • 进行安全审计、防止滥用与异常请求。

三、数据存储与保留

  • 应用内的设置与缓存数据保存在本地设备；卸载应用后，本地数据可能被删除。
  • 若您启用系统级云备份能力，数据可能按系统规则同步到您的云账户中。
  • 您主动提交的反馈信息会在必要期限内保存，用于问题处理与服务改进，之后将按业务需要删除或匿名化处理。

四、信息共享与第三方服务

我们不会出售您的个人信息。仅在以下必要场景下共享最小范围信息：

  • 第三方 API 服务：我们使用 GitHub API 获取公开的开源项目数据，使用第三方大模型 API 提供代码分析和摘要功能。
  • 应用商店支付与订阅管理：如 Apple App Store、华为应用内支付（IAPKit）等，用于处理订单与订阅状态。
  • 法律法规要求或为保障产品与用户安全所必需的情形。

五、数据安全

我们采取合理的技术和管理措施保护数据安全，包括但不限于 HTTPS 传输加密、访问控制与最小权限原则。但请理解，任何互联网传输或电子存储方式都无法保证绝对安全。

六、儿童隐私

CopoHub 可供不同年龄段用户使用。我们不会主动面向儿童收集超出必要范围的个人信息。若监护人发现儿童在未经同意情况下提交了个人信息，请联系我们处理。

七、政策更新

我们可能根据产品迭代与法规要求更新本政策。更新后版本将在本页面发布并生效。建议您定期查阅以了解最新内容。

八、联系我们

如您对本隐私政策有疑问，可通过客服邮箱联系我们：youdroid2048@gmail.com

Privacy Policy

We value your privacy and data security. This policy applies to the CopoHub (GitHub Explorer) app and its landing page, explaining how we handle information related to you.

1. Information We Collect

To provide core features, we may process the following information:

  • Local cache data: Such as your app settings, theme preferences, and cached GitHub data, mainly stored locally on your device.
  • Feedback information: When you actively submit feedback, we receive the content you fill in, along with optional contact details, client type, and app version.
  • Subscription and transaction information: When you purchase a Pro membership, the relevant order status is returned by the app store platform and used to verify and restore membership benefits.
  • Necessary technical information: When requesting AI analysis and data interfaces, the server may record basic logs (such as request time and interface status) for operations and security purposes.

2. Purpose of Information Use

We use information only to the extent necessary, for:

  • Ensuring core functions such as data retrieval and AI analysis operate normally;
  • Handling feedback, locating issues, and continuously improving the product experience;
  • Completing membership activation, subscription restoration, and related service support;
  • Conducting security audits and preventing abuse and abnormal requests.

3. Data Storage and Retention

  • Settings and cache data in the app are stored on the local device; local data may be deleted after uninstalling the app.
  • If you enable system-level cloud backup, data may be synced to your cloud account according to system rules.
  • Feedback you actively submit will be retained for the necessary period for issue handling and service improvement, then deleted or anonymized as needed.

4. Information Sharing and Third-Party Services

We do not sell your personal information. We share minimal information only in the following necessary scenarios:

  • Third-party API services: We use the GitHub API to obtain public open-source project data, and third-party large model APIs to provide code analysis and summary features.
  • App store payments and subscription management: Such as Apple App Store and Huawei in-app purchases (IAPKit), used to process orders and subscription status.
  • As required by laws and regulations, or as necessary to protect product and user security.

5. Data Security

We take reasonable technical and management measures to protect data security, including but not limited to HTTPS transmission encryption, access control, and the principle of least privilege. Please understand that no method of internet transmission or electronic storage can guarantee absolute security.

6. Children's Privacy

CopoHub is available to users of all ages. We do not actively collect personal information from children beyond what is necessary. If a guardian discovers that a child has submitted personal information without consent, please contact us.

7. Policy Updates

We may update this policy based on product iterations and regulatory requirements. Updated versions will be published and take effect on this page. We recommend checking periodically for the latest content.

8. Contact Us

If you have questions about this privacy policy, please contact us at: youdroid2048@gmail.com
